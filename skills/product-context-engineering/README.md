# Skill: Product Context Engineering

## O que é

Product Context Engineering é a metodologia ZRO para construir produtos digitais com foco, coerência e velocidade.

Ela define como pensar, documentar e executar antes de escrever qualquer linha de código.

## Por que existe

A maioria dos projetos falha por falta de contexto, não por falta de código.

Código sem visão gera retrabalho.
Retrabalho gera custo.
Custo mata projetos.

Esta skill elimina o retrabalho na origem.

## Quando usar

- Sempre que iniciar um novo produto
- Sempre que iniciar um novo módulo
- Sempre que a IA for implementar uma tarefa

## O que contém

| Arquivo | Conteúdo |
|---------|----------|
| `SKILL.md` | Metodologia completa — leia antes de tudo |
| `templates/` | Modelos vazios para preencher em cada projeto |
| `checklists/` | Verificações por momento (novo projeto, antes de codar, antes de deploy) |
| `prompts/` | Instruções para a IA executar com a metodologia |

## Regra de ouro

> Nenhum código antes da documentação estar completa e revisada.
